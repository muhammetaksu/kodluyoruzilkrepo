# Kodluyoruz Ilk Repo

## Installation
-----
Oncelikle projeyi clonelayin.
```
git clone https://github.com/muhammetaksu/kodluyoruzilkrepo.git
```
## Usage
-----
Projeyi cloneladiktan sonra Visual Studio Code programinda aciniz.

Linux icin:
```
cd kodluyoruzilkrepo
code .
```
